# Paleta de Colores - PetVilla

## Colores Principales

### Primary (Verde Esmeralda)
- **HSL**: `142 76% 36%`
- **Uso**: Botones principales, enlaces, elementos destacados
- **Significado**: Naturaleza, salud, cuidado, confianza

### Secondary (Azul Cielo)
- **HSL**: `199 89% 48%`
- **Uso**: Elementos secundarios, información complementaria
- **Significado**: Confianza, calma, profesionalismo

### Accent (Coral/Naranja)
- **HSL**: `24 95% 53%`
- **Uso**: Elementos de énfasis, llamadas a la acción secundarias
- **Significado**: Energía, diversión, calidez

## Colores de Estado

### Success (Verde)
- **HSL**: `142 71% 45%`
- **Uso**: Confirmaciones, estados exitosos

### Warning (Amarillo Dorado)
- **HSL**: `45 93% 47%`
- **Uso**: Advertencias, calificaciones, alertas

### Destructive (Rojo)
- **HSL**: `0 84% 60%`
- **Uso**: Errores, acciones destructivas

## Utilidades CSS

### Gradientes
- `.gradient-primary` - Gradiente verde
- `.gradient-accent` - Gradiente naranja
- `.gradient-warm` - Gradiente verde a azul
- `.text-gradient` - Texto con gradiente

### Sombras
- `.shadow-primary` - Sombra verde suave
- `.shadow-accent` - Sombra naranja suave

## Mejores Prácticas

1. Usar `primary` para acciones principales y elementos clave
2. Usar `secondary` para información complementaria y elementos secundarios
3. Usar `accent` para destacar elementos importantes pero no críticos
4. Usar opacidades (`/10`, `/20`, etc.) para fondos suaves
5. Los gradientes añaden dinamismo sin ser distractores
